A condição if

É uma condição "se", é usada para verificar se uma condição é verdadeira ou falsa. Se for verdadeira, ela atua, se for falsa, a outra condição atua

A condição else

É uma condição "senão", se o if for falso, ela atua

Dê exemplos!

  • Eu quero ligar a TV, "se" eu pressionar o botão ela liga, "senão" ela ficará desligada
  • "Se" o tempo estiver bom, eu irei sair, "senão", ficarei em casa
  • "Se" em um dado eu tirar 5, eu ganho, "senão" eu perco
